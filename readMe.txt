The following application was create by Lonwabo Tsipa.

Compiler used: Microsoft Visual Studio 2015
Operating system: Windows 10

Building the solution:
Open the application as a console application in visual studio and run the application, alternatively you can run the BinaryFileReader.exe application in the bin folder under the debug folder.


The application handles a bookstore inventory of their stock and how many books are brought in by the supplier.
The bookstore stores their data in binary files (.dat).  They are responsible for storing data and if there are any 
duplicated, it is audited by a log file on the Log folder in the project file. Users can then delete the files specified as duplicates to save up space.
